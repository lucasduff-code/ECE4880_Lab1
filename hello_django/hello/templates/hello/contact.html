Website Operation Manual

Power Button

The power button is used to togle the LCD screen on the temperature sensor. When you press the power button, the lcd screen on the temperature probe will turn on and display the most recent temerature reading. As soon as you release the power button, the lcd screen will turn off. Please note that this will only work if the temperature probe is powered on.

Notifications Box

The notifications box is used to set the minimum and maximum thresholds for acceptable temperatures. If the temperature exceeds the range you have defined. A text notification will be sent to the phone number listed in the box. Please note that you must hit the save button for any of your changes to take effect.

Ceclius to Farenheight Toggle Switch

The celcius to farenheight toggle switch can be used to toggle the website between celcius and farenheight.
